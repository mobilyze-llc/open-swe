# Project Documentation Index

Fork note: this repo is a thin fork of `langchain-ai/open-swe`. Docs conventions
here apply **additively only** — upstream-owned files (root `CLAUDE.md`,
`AGENTS.md`, `docs/CUSTOMIZATION.md`, `docs/INSTALLATION.md`) are never
restructured or renamed in the fork, and root `CLAUDE.md` carries no
`@docs/README.md` include for the same reason. Naming below applies to
fork-authored docs.

## Upstream docs

- `docs/CUSTOMIZATION.md` [UPSTREAM] — extending Open SWE (sandbox providers, tools, middleware, graphs).
- `docs/INSTALLATION.md` [UPSTREAM] — installing and running Open SWE.

## Point-in-time artifacts

- `docs/plans/` — decisions, plans, reviews, handoff-adjacent artifacts. Date-prefixed (`YYYY-MM-DD-slug.md`), frozen once written. Current decisions of record: `2026-07-21-agent-definition-convention.md` (the additive `agents/` convention and its fork-seam boundary).

## Archive

- `docs/archive/` — superseded fork-authored docs. Kept for history. Never delete; never edit.
